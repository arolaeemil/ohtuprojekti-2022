## Setup

1. Clone the [repo](https://github.com/immone/ohtuprojekti-2022/tree/dev)
```bash
git clone git@github.com:immone/ohtuprojekti-2022.git
````
2. Install dependencies
```bash
poetry install
````
3. Build
```bash
poetry run python3 src/build.py
````
